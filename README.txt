Extract colors from woocommerce products and show similar color products.

== Description ==

* Select a product image in the woolense tab
* Color will be extracted automatically
* Similar color products will be shown automatically

Note: You must use transparent background images to make it work perfectly.If you want to avoid this problem and want to pick any color from any image, upgrade to pro.

== Installation ==


1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
